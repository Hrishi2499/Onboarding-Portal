Welcome to the AU Management System's Onboarding Portal

Features

Candidates Display

This page displays the list of available candidates for onboarding. The candidates details are displayed and onboard button takes user to create onboard page. A search option is provided with various search parameters.

Candidates
Onboardees Display

This page displays the list of onboardees. The details of onboarding process are displayed and edit button takes user to update onboard page, to update some details. The delete button deletes the onboard process. The candidate becomes available for onboarding again on the candidates displaypage. A search option is provided with various search parameters.

Onboardees
Analytics

This page displays three graphs showing the onboarding trends across location, hiring managers and skills across the years.

Analytics
Logs

This page displays the logs of create, update and delete operations on the onboards and the users who made them. A search option is provided to search across various parameters.

Logs
